# Ink Cream

一个优雅的 [Ink](https://github.com/InkProject/ink) 博客主题。

设计灵感来自于 Astro 编写的 [fuwari](https://github.com/saicaca/fuwari) 主题。

## 特性

- 简洁优雅的设计和现代化界面
- 响应式布局
- 使用 Vite 构建，支持快速开发和热更新。本身也是 Vite 开发 Ink 主题的示例
